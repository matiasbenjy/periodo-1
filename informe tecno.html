Tablas

La etiqueta <table> permite al programador almacenar datos en filas y columnas.
<tr>: añade una columna a la tabla.
<td>: añade una fila a la tabla.

Contenedores (divs)

La etiqueta <div> permite almacenar objetos y elementos en un contenedor. Estos pueden tener atributos y caracteristicas.

Administración de dominios

Para subir un archivo al dominio, primero se selecciona el repositorio al que se va a subir el archivo.

Luego selecciona el botón "Agregar archivo", que desplegará un menú, y luego se selecciona la opción "Subir archivos".
Y por último, se seleccionan los archivos que se desean subir o se arrastran al campo.
NOTA: Si el archivo que va a subir ya existe, github se encarga de verificar que cambios tiene el archivo anterior con el actual, y genera versiones de los mismos.
